Principais tipos de Formatação

Negrito / Destaque

E posivel fazer um termo em Negrito utilizando a tag <b>(não Semantica)

Ja nesse paragrafo teremos um Termo em destaque usando a tag STRONG (semantica).

Itálico / ênfase

Utilizando a tag I posso criar uma palavra em Itálico (Não semantico).

Nesta frase, existe um termo em enfase utilizando a tag EM (semantica).

Texto marcado

E possivel criar um texto marcado utilizando a tag MARK

PENSA MARK 👉😠👈

Texto grande e pequeno

Nesse paragrafo existe um Texto grande e um Texto pequeno (Utilizando a tag big e small) ps: a tag big esta obsoleta

Texto Deletado

Podemos marcar um texto como excluido, para indicar que não deve ser considerado, mesmo que lido

Texto Inserido

E possivel marcar um texto como algo inserido, para representar que ele foi adicionado depois Utilizando a tag INS

Tambem existe a Tag U, para sublinhar, porem ela pode ficar obsoleta por isso utilize o Ins (U não e semantico)
